## üìñ Tentang Project

**BarisKode** adalah simulasi Landing Page untuk sebuah kursus pemrograman online. Tujuan utama dari project ini adalah melatih kemampuan **slicing design to code** yang presisi, rapi, dan responsif menggunakan teknologi modern tanpa framework JavaScript yang berat (Vanilla HTML + Tailwind).

Project ini menandai transisi saya dari CSS konvensional ke penggunaan **Tailwind CSS** secara menyeluruh.

## ‚ú® Fitur & Implementasi Teknis

Dalam project ini, saya menerapkan beberapa konsep kunci:

### 1. Advanced Layout dengan CSS Grid & Flexbox
* Menerapkan **CSS Grid** untuk tata letak kartu yang kompleks (Grid System).
* Menggunakan konsep `span` dan `grid-cols` untuk membuat layout yang dinamis.
* Menggunakan **Flexbox** untuk pengaturan navigasi (Navbar) dan penyelarasan vertikal/horizontal.

### 2. Styling dengan Tailwind CSS
* **Color Palette:** Menggunakan skema warna profesional (`Slate` untuk netral & `Blue` untuk primary) untuk membangun visual hierarchy yang baik.
* **Interactivity:** Menambahkan efek `hover`, `group-hover`, dan `transition` untuk pengalaman pengguna yang halus.
* **Glassmorphism:** Efek *backdrop-blur* pada Sticky Navbar.

### 3. Responsiveness
* Mobile-First approach.
* Penyesuaian layout otomatis dari layar Mobile (`block/grid-cols-1`) ke Desktop (`flex/grid-cols-3`).

## üß† Pembelajaran Utama (Key Takeaways)

Selama pengerjaan project ini, saya memperdalam pemahaman tentang:

1.  **Grid vs Flexbox:** Memahami kapan harus menggunakan Grid (untuk struktur 2D yang kompleks) dan kapan menggunakan Flexbox (untuk penyusunan 1D).
2.  **Tailwind Workflow:** Efisiensi penulisan style langsung di HTML tanpa berpindah file CSS, serta penggunaan utility class untuk spacing (`p`, `m`) dan typography.
3.  **UI Consistency:** Pentingnya menjaga konsistensi warna dan jarak antar elemen (whitespace) agar desain terlihat profesional.
